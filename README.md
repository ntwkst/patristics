# patristics

Padres da Igreja — **Ante-Nicene Fathers** e **Nicene and Post-Nicene Fathers**
(ed. Philip Schaff, séc. XIX, **domínio público**), em JSON estruturado
(autor → obra → capítulo), texto em inglês.

Fonte do texto: [gregorycrane/nicenefathers](https://github.com/gregorycrane/nicenefathers) (TEI XML).
Servido via jsDelivr CDN para o app [apologeticae](https://github.com/ntwkst/apologeticae).

Cada arquivo `<vol>.json` é um array de nós no schema do leitor (Scriptorium).
